# Calcutta Node — AI Agent Team

112 specialized AI agents installed across 5 divisions.

## How to Use

### In OpenCode sessions
Mention any agent by name with `@`:
```
@Frontend Developer build me a new pricing table component
@SEO Specialist audit my meta tags and suggest improvements
@Security Architect review my auth middleware
@Content Creator write a blog post about web development services
@UI Designer review this page layout for responsiveness
```

### Switch primary agents
Use Tab key to cycle between: `build`, `plan`, `reviewer`, `deploy`

### Run automated workflows
```powershell
.\scripts\auto-workflow.ps1 -Workflow full-audit
.\scripts\auto-workflow.ps1 -Workflow security-audit
.\scripts\auto-workflow.ps1 -Workflow seo-audit
.\scripts\auto-workflow.ps1 -Workflow deploy-check
```

### GitHub Actions
Push to main/master triggers automated security, SEO, UI, and content checks.
